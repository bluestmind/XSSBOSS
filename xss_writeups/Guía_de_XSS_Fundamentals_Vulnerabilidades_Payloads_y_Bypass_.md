## ¿Qué es Cross-site scripting (XSS)?

Es una vulnerabilidad de seguridad web que le permite a un atacante comprometer las interacciones que los usuarios tienen con una aplicación vulnerable. Permite eludir la política de origen, que está diseñada para segregar diferentes sitios web entre sí. Estas vulnerabilidades, normalmente, permiten a un atacante acceder a cualquiera de los datos. Si el usuario víctima tiene acceso privilegiado dentro de la aplicación, entonces se podría obtener control total sobre toda la funcionalidad y los datos de la aplicación.
